# Diwali_sales_analysis_Python
Project Description

This project focuses on analyzing Diwali season sales data to understand customer purchasing behavior and identify key revenue-driving factors. Using Python-based data analysis techniques, the project transforms raw sales data into meaningful business insights through data cleaning, exploratory data analysis (EDA), and visualization.
The goal is to demonstrate how data analytics can support business decision-making during high-demand festive periods.
